# ✅ To-Do List Manager

A simple and responsive **To-Do List Manager** built using **HTML, CSS, and JavaScript**.  
This project helps users **add, complete, and delete tasks** with a clean and interactive UI.

---

## 🚀 Features
- ✍️ Add new tasks dynamically  
- ✅ Mark tasks as completed  
- ❌ Delete tasks easily  
- 🎨 Responsive, clean, and minimal UI  
- 🔥 Built using pure HTML, CSS, and JavaScript (no frameworks)
